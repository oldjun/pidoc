# 交互模式

### 进入交互模式

在控制台里输入`pi`，即可进入交互模式：

```shell
Welcome to Pi Programming Language!
>>
```

示例：
```shell
Welcome to Pi Programming Language!
>>
>> a = 1
>> b = 2
>> c = a + b
>> c
3
```

::: tip 提示
在交互模式下，输入遇到回车换行，那么这行代码就会被立刻解释执行。
:::

### 退出交互模式

在交互模式里，输入`exit()`即可退出：

```shell
Welcome to Pi Programming Language!
>>
>> exit()
```
