# 标识符

标识符用来命名变量。一个合法的标识符由一个或是多个字母(A~Z和a~z)、数字(0~9)、下划线(-)组成。

```ts
num = 123
text = 'hello world'
bool = true
list = ['abc', 123, true]
hash = {'k1': num, 'k2': text, 'k3': list}
```

::: tip 提示
请尽量采用有意义且简洁的英文单词命名，可以提升代码的可读性，同时请遵循以下命名规范：
- 变量命名：建议字母全部小写，多个单词用下划线分隔。例如：`my_name = 'pi'`
- 函数命名：同变量命名。
- 类命名：建议采用大驼峰法。例如：`class MyName {}`
